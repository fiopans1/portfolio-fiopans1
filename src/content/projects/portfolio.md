# Portfolio fiopans1

## Overview

This portfolio is the place where I present my profile, selected projects, and the way I approach software.

## What I focused on

- A modern UI without overloading the page
- A React + Vite structure that is easy to maintain
- Routing for projects, resume, and detail pages
- Clear content hierarchy for quick scanning

## Why this project matters

Unlike a static résumé, this site gives context around the projects and makes the work easier to understand.

## Highlights

1. Custom project detail pages powered by markdown content
2. Responsive layout with a simple visual language
3. Reusable project data model for overview and detail views
