# Portfolio fiopans1

## Overview

This portfolio is my digital presentation space: a place to explain who I am, what I build, and how I think about software and design.

## What I focused on

- A visual system that feels modern without being overloaded
- A React + Vite structure that is easy to maintain and extend
- Routing for projects, resume, and deeper case-study style details
- Clear content hierarchy so recruiters and collaborators can scan fast

## Why this project matters

Unlike a static résumé, this site gives context. It lets me turn projects into stories, show decision making, and present my work with a stronger product feel.

## Highlights

1. Custom project detail pages powered by markdown content
2. Responsive layout with minimal visual language
3. Reusable project data model for both overview and detail views
