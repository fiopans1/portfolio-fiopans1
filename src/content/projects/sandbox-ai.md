# Sandbox AI

## Overview

Sandbox AI is an interactive playground designed to centralize, configure, and evaluate multiple Large Language Models (LLMs) from different providers within a single interface, facilitating real-time experimentation.

## What I built

- Multi-provider chatbot interface supporting native integration with OpenAI, Anthropic, Google Gemini, and local providers (like Ollama)
- Dynamic configuration panel to tune key model parameters (temperature, top_p, max_tokens, system prompts) on the fly
- Side-by-side comparison setup to evaluate and benchmark responses from different LLMs simultaneously
- Persistent conversation history and secure handling of API credentials and keys

## Why this project matters

It demonstrates my ability to integrate and abstract multiple AI APIs, manage complex multi-model UI states, and build practical tools that optimize prompt engineering and model selection workflow.

## Highlights

1. **Provider Abstraction:** Modular architecture allowing seamless integration of new LLM providers with minimal friction
2. **Granular Control:** Real-time hyperparameter adjustment to instantly measure shifts in model intelligence and behavior
3. **Developer Experience (DX) Focus:** Engineered for rapid testing and iteration without switching platforms or writing boilerplate code
