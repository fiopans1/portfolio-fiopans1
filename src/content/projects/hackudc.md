# HackUDC 2024

## Overview

HackUDC 2024 was a fast-paced build where the goal was to automate code analysis and transform technical metrics into readable reports.

## What I built

- A Python automation flow around code analysis tooling
- HTML reporting outputs with clear visual summaries
- CI automation through GitHub Actions
- Notification delivery through email workflows

## Why this project matters

Hackathons reward speed, but the interesting part is shipping something coherent under pressure. This project captures that ability to move quickly without losing structure.

## Highlights

1. Automation-first mindset from the start
2. Useful output for developers, not just raw metrics
3. Strong combination of scripting, reporting, and CI/CD
