# HackUDC 2024

## Overview

HackUDC 2024 was a hackathon project focused on automating code analysis and turning the results into readable reports.

## What I built

- A Python automation flow around Codee analysis
- HTML reports with visual summaries
- CI automation through GitHub Actions
- Email delivery for generated reports

## Why this project matters

It shows that I can build and ship quickly under pressure without losing structure.

## Highlights

1. Automation-first mindset from the start
2. Useful reports instead of raw analysis output
3. Strong combination of scripting, reporting, and CI/CD
